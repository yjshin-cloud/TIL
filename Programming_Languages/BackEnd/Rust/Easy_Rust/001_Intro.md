## 🎯 러스트 언어 특징 정리

- **안전성(Safety):** 메모리 관리 오류를 방지.
- **성능(Performance):** C/C++에 버금가는 속도.
- **동시성(Concurrency):** 안전한 멀티스레드 프로그래밍 지원.
- **컴파일 타임 검사:** 잘못된 코드 실행 전 오류 감지.
